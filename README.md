# Dino-Game
A fun, browser-based clone of the classic Google Chrome Dino T-Rex game.
Built with HTML, CSS, and JavaScript, this version features:

Dynamic obstacles 🌵

Scoring system 🎯

Game over detection 💀

Start & Retry buttons 🔁

Day-Night cycle (every 30 sec) 🌞🌙

Responsive design 💻📱
